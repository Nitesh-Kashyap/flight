# Nitesh Flight
Flight booking web application.

### Concept Used
* STI - Single Table Inheritance
* ActiveModel has_secure_password - Authentication

### Gems Used
* letter_opner - to view mails in browser
* bcrypt - to authenticate
* faker - dummy data
